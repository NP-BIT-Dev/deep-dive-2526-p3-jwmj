Dit is de reflectie op de deepdive 2526.

In hoeverre is the PVE gerealiseerd?
Het is voltooid, in vrij redelijk termen vindt ik.

Welke eisen zijn voldaan?
Vrij wel alles

Waarom?
Opdracht gever had zelf al weinig eisen.


Wat is er gerealiseerd?
Wij hebben een functionele website gemaakt met een meestal werkende API.

Wat is de kwaliteit van het prototype?
Het prototype was van redelijk goeie kwaliteit na mijn persoonlijke mening.

Wat ontbreekt nog?
Een Admin paneel en logo.


Wat ging er goed?
Samenwerking, werk verdeling en zelf standigheid.

Wat ging er fout?
Technische storingen, hier en daar, slechte informatie van client en mijn eigen toevoeging aan de groep.

Hoe was de communicatie?
De communicatie was redelijk goed, we waren weleens afgeleid maar dat moet kunnen.

Rol binnen de groep.
Ik was de designer van de groep, dit betekent dat ik de wireframe ontworpen heb en een logo (dat wij niet gebruikt hebben) en een huisstijl ik heb ook een pagina van de website gemaakt, die later verfijnd is.


Wat heb ik geleerd?
Ik heb geleerd hoe je HTML in Javascript kunt schrijven en wat components zijn, daarnaast heb ik nu meer ervaring met Tailwind CSS.

Wat neem je mee naar het volgende project?
Terwijl ik vrees dat het meeste van deze lessen al verloren zullen zijn bij de tijd dat het volgende project rond komt, zal ik dankzij de ervaring die ik hier verkregen heb makkelijker uit te voeten kunnen met geadvanceerde features van Javascript en ook met Tailwind.

Wat zou jij persoonlijk anders doen?
Ik zou graag met meer kennis over Javascript van start gaan zodat ik nuttiger ben voor de groep, daarnaast is er eigenlijk niks dat ik anders zou kunnen voor dit project.
