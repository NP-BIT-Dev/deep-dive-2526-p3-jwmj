## 02 PvE

**In hoeverre is het PvE gerealiseerd?**
*   Er is een basis gelegd voor de frontend en een gedeelte backend.

**Welke eisen wel/niet gehaald?**
*   **Must Have - Mobiel-vriendelijk:** Gehaald. De componenten gebruiken Tailwind classes (bijv. `col-span-full`, `w-20`).
*   **Could Have - Animaties:** Gehaald. De 404-pagina bevat uitgebreide CSS-animaties (met hulp van een codepen project).
*   **Should Have - Nieuwsrubriek:** Werkt volledig dankzij de API van Julian.

**Waarom?**
*   De focus lag op het opzetten van herbruikbare componenten.
